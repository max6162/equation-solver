Типы поддерживаемых уравнений

1. Квадратные уравнения

Форма: ax² + bx + c = 0 (где a ≠ 0)

Пример: 1x² - 5x + 6 = 0

Корни: x₁ = 3, x₂ = 2

2. Линейные уравнения

Форма: ax + b = 0 (где a ≠ 0)

Пример: 2x - 4 = 0

Корень: x = 2

3. Кубические уравнения

Форма: ax³ + bx² + cx + d = 0 (где a ≠ 0)

4. Биквадратные уравнения

Форма: ax⁴ + bx² + c = 0

5. Системы линейных уравнений 2x2

Форма:

a₁x + b₁y = c₁ a₂x + b₂y = c₂

Форматы ввода

Тип Примеры Описание

Дроби 1/2, 3/4, -2/3 Формат: числитель/знаменатель

Примеры решения

Пример 1: Квадратное уравнение с двумя корнями

Уравнение: x² - 5x + 6 = 0

Коэффициенты: a = 1, b = -5, c = 6

Решение:

D = b² - 4ac = 25 - 24 = 1 x = (-b ± √D) / (2a) x = (5 ± 1) / 2 x₁ = 3 x₂ = 2

Пример 2: Квадратное уравнение с комплексными корнями

Уравнение: x² + 2x + 5 = 0

Коэффициенты: a = 1, b = 2, c = 5

Решение:

D = 4 - 20 = -16 < 0 Уравнение имеет комплексные корни: x₁ = -1 + 2i x₂ = -1 - 2i

Пример 3: Линейное уравнение

Уравнение: 3x - 9 = 0

Решение:

3x = 9 x = 3

Пример 4: Система уравнений 2x2

Система:

2x + 3y = 8 x - y = 1

Решение: x = 2.2, y = 1.2

Теорема Виета

Если x₁ и x₂ - корни уравнения ax² + bx + c = 0, то: x₁ + x₂ = -b/a x₁ · x₂ = c/a

Полезные советы

Для квадратного уравнения коэффициент a должен быть ненулевым (a ≠ 0)
Используйте точку (.) для десятичных чисел, не запятую
Для дробей используйте формат числитель/знаменатель без пробелов
Нажмите на примеры уравнений для быстрой загрузки
История решений сохраняется автоматически в памяти браузера
Кнопка "Генерировать" создаёт случайные уравнения для практики
Графики интерактивны - перетащите для панорамирования, прокрутите для масштабирования
Решение можно скопировать или экспортировать после получения результата

Часто задаваемые вопросы

Где хранится история решений?

Ответ: История сохраняется в localStorage браузера. Она будет доступна, пока вы не очистите данные браузера. Максимально хранится 50 последних решений.

Как скачать решение?

Ответ: После получения результата нажмите кнопку "PDF/Текст" для загрузки решения как текстового файла.

Работает ли без интернета?

Ответ: Да! Это полностью локальное приложение. Всё работает оффлайн после загрузки страницы.

Могу ли я использовать очень большие числа?

Ответ: Да, но результаты будут округлены до 6 знаков после запятой для удобства чтения.

Как поделиться решением?

Ответ: Нажмите кнопку "Поделиться" и скопируйте ссылку или поделитесь в социальных сетях.

Поддерживаются ли иррациональные коэффициенты?

Ответ: Введите десятичное приближение числа. Например, вместо √2 используйте 1.414
